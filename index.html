[ SYSTEM_DOCUMENTATION // USER_GUIDE ]

Welcome to Spiokoks Terminal Workspace

Deploy this framework using the following control matrices:

1. File Ingestion Matrix

  • Open PDF: Pulls a single localized PDF document into the core render pipeline.
  • Combine Files: Aggregates multiple asset nodes and bundles them into a master document stream.

2. Page & Node Mutation

  • Organize Pages: Opens the visual deck array. Drag individual thumbnails to restructure index sequence. Use overlay buttons to rotate orientation or clear page objects entirely. You can also append blank layouts or inject fresh documents directly into the active matrix deck. Click Apply Reorganization to bake configurations.
  • Split PDF: Chops the loaded target file into block increments based on your specified sequence volume chunk constraints.

3. Workspace Annotation Layover

  • Edit PDF (Add Text): Assigns standard typographic objects onto selected canvas zones. Highlight item grids to reposition layout, alter scale bounds via corner anchors, or alter structural string metrics inline.
  • Attach File to Page: Embeds image files directly over target canvas coordinates using localized scale structures.
  • Ensure you commit your modifications by executing the Apply Changes routine to overlay alterations securely.

4. Compiling, Resizing & Document Extraction

  • Document Resizing: Once a document is loaded, a size control cluster appears in the top-right of the header, labeled DOC SIZE.
    • A unit selector next to the label lets you switch between inches, centimeters, millimeters, points, and pixels at any time. Switching units re-displays your current size in the new unit without changing the actual document dimensions — all size math happens internally in a fixed unit and only converts for display, so nothing drifts no matter how many times you switch.
    • Enter target Width and Height in the selected unit, or set a target Resolution in DPI — all three fields stay mathematically linked. (Resolution is always dots-per-inch regardless of which size unit is selected, and also governs the Pixels unit's conversion, since pixel dimensions depend on DPI.)
    • The 🔗 icon between Width and Height is the Aspect Ratio Lock (linked by default): editing either dimension automatically recalculates the other to preserve proportions. Click it to unlink and set Width/Height independently — note that resizing with this unlinked will stretch the page content to fit the new dimensions, since content is rescaled to exactly fill the new page size.
    • The 🔗 icon between Height and Resolution is the Resolution Lock: with it linked, changing the physical inches or the DPI keeps the underlying pixel dimensions constant (raising DPI shrinks the physical size for the same pixel count, and vice versa).
    • Click RESIZE to bake in the new dimensions. This builds a fresh document with every page re-embedded and scaled into the specified page size, then refreshes the live preview automatically.
  • Export PDF: Compiles layered node logic safely into a finalized, download-ready portable data format document.

5. Adaptive Preview & Navigation Matrix

The render pipeline auto-detects your host hardware and switches preview engines accordingly, since not every browser can display a PDF the same way.

  • Desktop Terminals (Windows / macOS / Linux / ChromeOS): Uses the browser's native inline PDF plugin for maximum performance, including its built-in zoom, print, and text-selection controls.
  • Android / Tablet / Unclassified Devices: Switches to the internal canvas-rendering engine, since mobile browsers can't reliably display PDFs inside an embedded frame. Pages stream into a continuous vertical scroll, like a native mobile reader.
  • Page Thumbnail Rail: On mobile mode, a scrollable strip of page thumbnails runs along the left edge of the viewer. Tap any thumbnail to jump straight to that page; the rail auto-highlights and follows your current position as you scroll.
  • Floating Navigation Bar: Also mobile-only — a control cluster docked to the bottom of the screen shows your current page position and provides Zoom In / Zoom Out controls. It stays anchored to the true bottom edge of the display in both portrait and landscape, regardless of browser chrome.
  • Pinch-to-Zoom: On touchscreens, use a two-finger pinch gesture directly on the document to zoom smoothly. Release to lock in the new zoom level at full sharpness.
